appleTV-slideshow is [the start of] a jQuery plugin that attemps to recreate the "Float" slideshow from the Apple TV (launched September 2010) using smooth webkit animations (webkit/chrome/safari only) of photos pulled from Flickr.

I just guessed the values of how to animate the photos upwards, the basic idea is here, but getting real accuracy would require a lot more work, so I'm putting this project here as a starting point.

The real slideshow has photos animating individually with at least 3 different speeds, here I have them all animating the same.

This was done for fun as an experiment in a few hours, though could be used practically with some work as a slideshow within a div container on a page. The following TODOs represent some more work I'd like to do to make the effect more like the original.
